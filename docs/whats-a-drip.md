Drips üíß is an Ethereum protocol for streaming and splitting tokens. It allows users and web3 apps to stream and split funds by the second, enabling continuous settlement for use cases like contributor payments, vesting and subscription memberships.

## Status of Drips v2

As of March 2023, Drips V2 is currently deployed as a Beta preview release on Ethereum Mainnet.
